## Scripts
1. system_info.sh  
Displays system details like kernel, user, uptime.
2. package_check.sh  
Checks if Python is installed.
3. disk_audit.sh  
Shows directory size and permissions.
4. log_analyzer.sh  
Counts errors in log files.
5. manifesto.sh  
Generates open source philosophy text.
## How to run
chmod +x *.sh
./system_info.sh
./package_check.sh
./disk_audit.sh
./log_analyzer.sh /var/log/syslog
./manifesto.sh
